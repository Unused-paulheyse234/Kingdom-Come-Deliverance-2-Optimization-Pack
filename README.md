## 🎮 Recommended Game Settings

These settings are a good place to start:

- Display Mode: Fullscreen
- Resolution: Native or one step lower if needed
- V-Sync: Off unless you see screen tearing
- Shadow Quality: Medium or Low
- Texture Quality: High if you have enough VRAM
- Crowd Density: Medium
- Motion Blur: Off
- Depth of Field: Low or Off
- Post Processing: Low or Medium
- Anti-Aliasing: Use the lighter option if performance is weak

If the game feels slow, lower shadows first. Shadows often have a big cost.

## 🧪 Upscaling Options

If your system supports upscaling, try one of these:

- DLSS on NVIDIA cards
- FSR on AMD and many other GPUs
- XeSS on supported Intel and other cards

Use a balanced mode if you want better speed with good image quality. If the game still feels heavy, move to performance mode.

If the game does not start:

- remove the new config files
- return to your backup
- lower shadows and crowd settings

- restart the game after each change

If the screen looks wrong:

- switch back to native resolution
- turn off aggressive sharpening
- try a different upscaling mode
- reset the game video settings
